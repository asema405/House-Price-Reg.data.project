#  Описание проекта
Данный проект представляет собой веб-приложение на базе Flask для прогнозирования стоимости жилья. Прогноз строится на основе модели машинного обучения (линейная регрессия), которая анализирует входные данные о недвижимости....
# Цель проекта 
Разработка веб-интерфейса, который позволяет пользователю быстро получить оценку стоимости дома, используя обученную математическую модель...
# Используемые технологии  
* **Python**
* **Flask**
* **Pandas**
* **Scikit-learn**
* **Matplotlib**
* **HTML / CSS**
# Структура проекта
---
* `app.py` — основной файл веб-приложения (сервер Flask)
* `train.py` — скрипт для обработки данных, обучения модели и создания файлов `.pkl`
* `predict.py` — отдельный скрипт с логикой предсказания
* `house_price_regression_dataset.csv` — датасет, на котором обучается модель
* `model.pkl` — файл сохраненной обученной модели
* `features.pkl` — файл со списком признаков (колонок), необходимых для корректного ввода данных
* `templates/index.html` — HTML-шаблон пользовательского интерфейса
* `static/plot.png` — график зависимости реальных цен от предсказанных

# Использование
---
Пользователь вводит следующие параметры недвижимости для расчета стоимости:

* **Square_Footage** — Общая площадь жилья в квадратных футах.
* **Num_Bedrooms** — Количество спальных комнат.
* **Num_Bathrooms** — Количество ванных комнат.
* **Year_Built** — Год постройки дома.
* **Lot_Size** — Размер земельного участка.
* **Garage_Size** — Вместимость гаража (количество машин).
* **Neighborhood_Quality** — Индекс качества района.

После ввода всех данных и нажатия кнопки **Predict**, система автоматически рассчитывает и отображает прогнозируемую стоимость жилья на экране.
